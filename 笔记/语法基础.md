## 基本语法

```go
package main

import "fmt"

func main() {
   fmt.Println("Hello 世界!")
}
```

package关键字声明了是当前go文件属于哪一个包，入口文件都必须声明main包，入口函数是main函数，再自定义包和函数时应当尽量避免与之重复

### 包

在Go中，程序是通过将包连接在一起构建的。go中进行导入的基本单位是一个包，而不是.go文件。包内共享所有变量，常量，以及所有定义类型。报的命名风格建议都是小写字母，并且要尽量简短

### 可见性

- 名称大写字母开头，即为公有类型/变量/常量
- 名字小写或下划线开头，即为私有类型/变量/常量

### 内部包

go中约定，一个包内名为internal包为内部包，外部包将无法访问内部包中的任何内容，否则编译不通过

### 运算符

go语言中没有将~作为取反运算符，而是复用了^符号，当两个数字使用^时，如a^b，就是异或运算符，支队一个数字使用时，^a就是取反运算

### 整形字面量

为了便于阅读，允许使用下划线进行数字划分，但是仅允许在前缀符号之后和数字之间使用

``` go
24 // 24
024 // 24
2_4 // 24
0_2_4 // 24
10_000 // 10k
100_000 // 100k
0O24 // 20
0b00 // 0
0x00 // 0
0x0_0 // 0
```

### 函数

Go中的函数声明通过func关键字来声明

不过Go中的函数有两个不同的点，第一个是参数类型后置

``` go
func Hello(name string) {
  fmt.Println(name)
}
```

第二个不同的点就是多返回值，而且可以带名字

```go
func Pos() () (x, y float64) {
    ...
}
```

GO中是没有三元表达式的





## 数据类型

### 整形

Go 中为不同位数的整数分配了不同的类型，主要分为无符号整型与有符号整型。

| 序号      | 类型和描述                                                 |
| --------- | ---------------------------------------------------------- |
| `uint8`   | 无符号 8 位整型                                            |
| `uint16`  | 无符号 16 位整型                                           |
| `uint32`  | 无符号 32 位整型                                           |
| `uint64`  | 无符号 64 位整型                                           |
| `int8`    | 有符号 8 位整型                                            |
| `int16`   | 有符号 16 位整型                                           |
| `int32`   | 有符号 32 位整型                                           |
| `int64`   | 有符号 64 位整型                                           |
| `uint`    | 无符号整型 至少 32 位                                      |
| `int`     | 整型 至少 32 位                                            |
| `uintptr` | 等价于无符号 64 位整型，但是专用于存放指针运算，用于存放死 |

### 浮点型 

主要分为单精度浮点数和双精度浮点数

| 类型      | 类型和描述           |
| --------- | -------------------- |
| `float32` | IEEE-754 32 位浮点数 |
| `float64` | IEEE-754 64 位浮点数 |

### 派生类型

| 类型   | 例子                                                |
| ------ | --------------------------------------------------- |
| 数组   | `[5]int`，长度为 5 的整型数组                       |
| 切片   | `[]float64`，64 位浮点数切片                        |
| 映射表 | `map[string]int`，键为字符串类型，值为整型的映射表  |
| 结构体 | `type Gopher struct{}`，Gopher 结构体               |
| 指针   | `*int`，一个整型指针。                              |
| 函数   | `type f func()`，一个没有参数，没有返回值的函数类型 |
| 接口   | `type Gopher interface{}`，Gopher 接口              |
| 通道   | `chan int`，整型通道                                |

### 零值

官方文档中零值称为`zero value`，零值并不仅仅只是字面上的数字零，而是一个类型的空值或者说默认值更为准确。

| 类型                                 | 零值                         |
| ------------------------------------ | ---------------------------- |
| 数字类型                             | `0`                          |
| 布尔类型                             | `false`                      |
| 字符串类型                           | `""`                         |
| 数组                                 | 固定长度的对应类型的零值集合 |
| 结构体                               | 内部字段都是零值的结构体     |
| 切片，映射表，函数，接口，通道，指针 | `nil`                        |

### nil

`nil`类似于其它语言中的`none`或者`null`，但并不等同。`nil`仅仅只是一些引用类型的零值，并且不属于任何类型，从源代码中的`nil`可以看出它仅仅只是一个变量。

## 常量

### 初始化

常量的初始化需要用到const关键词，并且必须指定值，否则无法通过编译

批量声明可以通过()括起来提升可读性，可以存在多个（）达到分组的效果

```go
const (
   Count = 1
   Name  = "Jack"
)

const (
   Size = 16
   Len  = 25
)
```

### iota

iota是一个内置的常量标识符，通常用于表示一个常量中声明的无类型整数序数，一般都是在括号中使用

```go
const (
   Num = iota // 0
   Num1 // 1
   Num2 // 2
   Num3 // 3
   Num4 // 4
)
```

也可以

```go
const (
   Num = iota*2 // 0
   Num1 // 2
   Num2 // 4
   Num3 // 6
   Num4 // 8
)
```

### 枚举

Go语言没有为枚举单独设计一个数据类型，可以通过自定义类型+const+iota实现

```go
type session uint8
const (
	Spring session = iota
	Summer
	Autumn
	Winter
)

```

### 变量

在go中的类型声明是后置的，变量的声明用var，格式为 var 变量名 类型名

声明多个相同类型变量时，可以只写一次类型

```go
func main() {
	var a, b, c, d = 1, 2, 3, 4
	fmt.Printf("a + b + c + d =%d", a+b+c+d)
}
```

当要声明多个不同的类型的变量时可以使用（）包裹，可以存在多个（）

```go
var (
  name    string
  age     int
  address string
)

var (
  school string
  class int
)
```

### 匿名

用下划线可以表示不需要某一个变量

比如os.Open函数有两个变量，我们只想要第一个，不想要第二个，可以按照下面这样写

``` go
file,_ := os.Open(readme.txt)
```

未使用的变量是无法通过编译的，不需要某一个变量时，就可以用下划线代替

## 循环控制

再Go 中，有且仅有一种循环语句：for，Go抛弃了while语句，for语句可以被当作while来使用

语句格式如下:

``` go
for init statement ;expression;post statement{
    
}
```

当只保留循环条件时，就变成了`while`。

```
for expression {
  execute statement
}
```

这是一个死循环，永远也不会退出

```
for {
  execute statement
}
```
### 切片

在Go中，数组是定长的数据结构，长度被指定后就不能改变。而切片是不定长，切片在容量不够时回自行扩容

数组在声明时长度应该是一个常量，不能是变量。不能在声明一个变量让后做为数组的长度值

初始化数组

```go
nums := [5]int{1, 2, 3}
```

也可以用编译器自动判断长度

```go
nums := [...]int{1, 2, 3, 4, 5}
```

### 切割

切割数组的格式为arr[startIndex,endIndex],切割的区间为左闭右开。且数组在切割后，就会变为切片类型

```go
nums := [5]int{1, 2, 3, 4, 5}

nums[:] // 子切片范围[0,5) -> [1 2 3 4 5]
nums[1:] // 子切片范围[1,5) -> [2 3 4 5]
nums[:5] // 子切片范围[0,5) -> [1 2 3 4 5]
nums[2:3] // 子切片范围[2,3) -> [3]
nums[1:3] // 子切片范围[1,3) -> [2 3]
```

若要将数组转换为切片类型，不带参数进行切片即可，转换后的切片与原数组指向的时同一片内存，修改切片会导致原数组内容变化

```go
func main() {
	var arr1 [5]int = [5]int{1, 2, 3, 4, 5}
	ints := arr1[:]
	ints[0] = 100
	fmt.Println(arr1)
}

打印结果
[100 2 3 4 5]

进程 已完成，退出代码为 0
```
